# Support

## Getting Help

### 1. Check the Documentation

Before opening an issue, please check:
- [Getting Started Guide](docs/getting-started.md)
- [FAQ](docs/faq.md)
- [Troubleshooting](docs/troubleshooting.md)

### 2. Search Existing Issues

Your question may already be answered:
- [Search Issues](https://github.com/coder4nix/XUIONE-WEBPLAYER-PUBLIC/issues?q=is%3Aissue)

### 3. Open a New Issue

If you still need help:

#### Bug Reports
Use the [Bug Report Template](https://github.com/coder4nix/XUIONE-WEBPLAYER-PUBLIC/issues/new?template=bug_report.md)

Please include:
- XUIONE WebPlayer version
- Operating system and browser
- Steps to reproduce the issue
- Expected behavior vs actual behavior
- Screenshots or error logs

#### Feature Requests
Use the [Feature Request Template](https://github.com/coder4nix/XUIONE-WEBPLAYER-PUBLIC/issues/new?template=feature_request.md)

Please include:
- Clear description of the feature
- Use case / why you need it
- Any examples or mockups

## Response Times

- **Critical bugs** (app not working): Within 24-48 hours
- **Normal bugs**: Within 1 week
- **Feature requests**: Reviewed monthly
- **Questions**: Community-driven

---

## What We Need From You

To help you faster, please always provide:

1. **Version number** (Settings → About)
2. **Browser and OS** (e.g., Chrome 120, Windows 11)
3. **Error messages** (from browser console: F12 → Console)
4. **Steps to reproduce** the issue
5. **Screenshots** if applicable

---

## Security Issues

If you discover a security vulnerability, please **do not** open a public issue.

Instead, please report it privately by creating a [Security Advisory](https://github.com/coder4nix/XUIONE-WEBPLAYER-PUBLIC/security/advisories/new).

---

## Community Guidelines

- Be respectful and constructive
- Search before posting duplicates
- Provide detailed information
- Be patient - this is a community-supported project
