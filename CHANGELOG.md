## [1.0.0] - 2026-03-04

### Initial Release

- Live TV with EPG (Electronic Program Guide) and channel management
- Movies & Series library with TMDB metadata integration
- List & Grid view with inline player for live channels
- Picture-in-Picture support for multitasking
- Multi-profile support with customizable settings
- Xtream Codes/XUI.ONE API and 1-Stream API support
- Built-in CORS proxy for seamless streaming
- Dark & Light mode with customizable themes
- Global search across all content
- Keyboard and remote control navigation
- Admin panel with branding, maintenance mode, and system stats
- Server-side license management with JWT validation
- Automated installer for Ubuntu, Debian, CentOS, Fedora, Arch, and Alpine
- Customer portal for subscription and billing management
- Self-Service and Managed Mode support
