# Frequently Asked Questions

## General

### What is XUIONE WebPlayer?
XUIONE WebPlayer is a professional web-based IPTV player that supports Xtream Codes API and M3U playlists.

### What do I need to use XUIONE WebPlayer?
- A valid XUIONE license key
- An IPTV subscription (Xtream Codes or M3U)
- A web browser or server to host the player

### Is XUIONE WebPlayer free?
No, XUIONE WebPlayer requires a commercial license. Different tiers are available (Basic, Pro, Enterprise).

## Installation

### Can I run XUIONE on a Raspberry Pi?
Yes! XUIONE is lightweight and runs well on Raspberry Pi 4 and similar devices.

### Does XUIONE work on mobile devices?
Yes, the web interface is fully responsive and works on smartphones and tablets.

### Can I install XUIONE on a NAS?
Yes, using Docker. Most NAS systems (Synology, QNAP) support Docker containers.

## Features

### Does XUIONE support EPG?
Yes, full Electronic Program Guide support with 7-day schedules.

### Can I use multiple IPTV providers?
Currently, XUIONE supports one provider at a time. Multi-provider support is planned.

### Does XUIONE support catch-up TV?
Yes, if your IPTV provider supports catch-up/timeshift.

## Troubleshooting

### Channels are not loading
1. Check your Xtream Codes credentials
2. Verify the server URL is correct
3. Check browser console for errors (F12)
4. Ensure your IPTV subscription is active

### Video not playing
1. Try a different channel
2. Check if your browser supports the video format
3. Try disabling hardware acceleration
4. Clear browser cache

### License validation failed
1. Verify your license key is correct
2. Check your internet connection
3. Ensure the domain matches your license

## License

### Can I transfer my license to another domain?
Yes, you can deactivate and reactivate on a different domain (within your tier limits).

### What happens when my license expires?
Lifetime licenses don't expire. The software will continue to work, but you won't receive updates.

### Can I get a refund?
Yes, within 14 days of purchase if the license hasn't been activated.
